## 🗺️ Full Roadmap (Step-by-Step)

### 🟢 Module 1: Networking Fundamentals
* **What is Networking?**
* **Types of Networks:** LAN, WAN, MAN.
* **OSI Model:** The backbone of network communication (🔥 High Importance).
* **TCP/IP Model:** Practical implementation.

### 🟡 Module 2: IP Addressing
* **What is an IP Address?**
* **IPv4 Structure:** Mastering the basics.
* **Subnetting:** The most critical skill (🔥 High Importance).
* **Private vs. Public IP:** Understanding NAT and address space.

### 🔵 Module 3: Switching
* **What is a Switch?**
* **MAC Address Table:** How switches learn.
* **VLANs:** Segmentation and security.
* **Trunking:** Carrying traffic between switches.
* **STP (Spanning Tree Protocol):** Introduction.
* *Note: Includes hands-on L2 switch lab demonstrations.*

### 🔴 Module 4: Routing
* **What is a Router?**
* **Static Routing:** Building manual paths.
* **Default Route:** Simplifying traffic flow.
* **Introduction to OSPF:** Dynamic routing basics.

### 🟣 Module 5: Troubleshooting
* **Ping & IP Troubleshooting:** Testing connectivity.
* **Link Down Problems:** Identifying physical layer issues.
* **VLAN/Gateway Issues:** Fixing common configuration errors.

### ⚫ Module 6: Real Field Knowledge (The "Diploma Wale" Edge)
* **LOS (Loss of Signal):** Identifying physical medium issues.
* **P2P vs. P2MP:** Understanding deployment topologies.
* **ISP Network Basics:** How internet service providers deliver bandwidth.
* **Field Engineer Work:** Daily workflows and technical challenges in the field.
