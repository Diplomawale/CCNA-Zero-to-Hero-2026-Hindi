# 🌐 What is a Network?

A network is a group of devices connected together to share:

- Data
- Internet
- Resources
- Communication

Examples of devices:
- Laptop
- Mobile
- Router
- Switch
- Server

---

# 📡 Simple Network Example

```text
Laptop → Router → Internet
```

---

# 🟢 What is LAN?

## 📖 Full Form
# Local Area Network

LAN is used to connect devices within a small geographical area.

---

# 📍 Where LAN is Used?

- Home WiFi
- Office Network
- School Lab
- College
- Cyber Cafe

---

# 🎯 LAN Real-Life Example

When your:
- Laptop
- Mobile
- Smart TV

are connected to the same WiFi router, that is called a LAN network.

---

# ✅ Features of LAN

| Feature | Description |
|---|---|
| Area | Small |
| Speed | High |
| Cost | Low |
| Ownership | Private |

---

# 📊 LAN Diagram

![LAN Diagram](../images/network/lan-diagram.png)

---

# 🧪 LAN Packet Tracer Lab

```text
PC1 → Switch → PC2
```

---

# 🔥 LAN Troubleshooting Example

Problem:
```text
PC1 cannot ping PC2
```

Possible causes:
- Cable disconnected
- Wrong IP address
- Switch port down

---

# 🔵 What is WAN?

## 📖 Full Form
# Wide Area Network

WAN connects multiple LAN networks over large geographical distances.

---

# 🌍 WAN Examples

- City-to-city connection
- ISP Network
- Bank branch connectivity
- Telecom backhaul network

---

# 🎯 WAN Real-Life Example

Suppose:
- Mumbai office connected to Delhi office

This connection is called WAN.

---

# 📡 ISP Example

Tower-to-tower wireless connectivity used by ISPs is also a WAN example.

---

# ✅ Features of WAN

| Feature | Description |
|---|---|
| Area | Large |
| Speed | Lower than LAN |
| Cost | Higher |
| Ownership | ISP / Telecom |

---

# 📊 WAN Diagram

![WAN Diagram](../images/network/wan-diagram.png)

---

# 🧪 WAN Packet Tracer Lab

```text
Router1 → Serial Link → Router2
```

---

# 🔥 WAN Troubleshooting Example

Problem:
```text
Branch office internet not working
```

Possible causes:
- WAN link down
- ISP issue
- Routing issue

---

# 🔴 What is Internet?

## 📖 Definition

The Internet is a global network that connects millions of networks worldwide.

---

# 🌎 Internet Examples

- YouTube
- Google
- WhatsApp
- Instagram
- Facebook

---

# 🎯 Internet Real-Life Flow

```text
Mobile → WiFi Router → ISP → Internet → Server
```

---

# 📊 Internet Diagram

![Internet Diagram](../images/network/internet-diagram.png)

---

# 🔥 How Internet Works?

1. Device sends request
2. Router forwards traffic
3. ISP carries data
4. Internet reaches server
5. Server sends response back

---

# 📊 LAN vs WAN vs Internet

| Feature | LAN | WAN | Internet |
|---|---|---|---|
| Full Form | Local Area Network | Wide Area Network | Global Network |
| Coverage | Small | Large | Worldwide |
| Speed | High | Medium | Depends |
| Cost | Low | High | Public |
| Example | Home WiFi | ISP Network | YouTube |

---

# 📡 Real Telecom Field Example

In ISP networks:

- Towers are connected through wireless links
- Routers carry WAN traffic
- Switches manage local LAN traffic

This is how real telecom and ISP infrastructure works.
